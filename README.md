# myshell
Linux 课程设计 简易shell实现

参考：
- https://github.com/xiabee/MyShell
- https://pan.baidu.com/s/1e0JlPAv-sdnOzUiLBKXCdw 提取码：m5pw

# 编译运行
```shell
git clone https://github.com/Erebua/myshell.git
cd ./myshell
gcc -o shell shell.c
./shell
```

# 目前支持的命令
- cd
- ls -al
- pwd
- whoami
- cat <file_name>

# 编译环境
gcc version 9.4.0 (Ubuntu 9.4.0-1ubuntu1~20.04.1)
